# User Manual

## Workflow
1. **Upload**: Ngarko PDF nëpërmjet API ose UI.
2. **Processing**: Sistemi ekstrakt të dhënat automatikisht.
3. **Review**:
   - Nëse **Confidence > 80%** dhe **Math Valid**: Statusi "Extracted".
   - Nëse **Confidence < 80%**: Statusi "Needs Review".
4. **Approval**: Përdoruesi klikon "Approve" ose "Reject" në dashboard.
